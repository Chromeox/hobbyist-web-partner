HobbyistSwiftUI

A modern iOS application for discovering and booking hobby classes

Get Started View on GitHub

Overview

SwiftUI Native

Built with 100% SwiftUI for iOS 16+, leveraging the latest Apple technologies and design patterns.

Supabase Backend

Powered by Supabase for real-time data sync, authentication, and PostgreSQL database with Row Level Security.

Secure Payments

Stripe integration for secure payment processing with support for Apple Pay and credit cards.

Features

For Students

  • Browse classes across 12+ categories
  • Smart filtering by location, price, and difficulty
  • Credit pack system for better value
  • Achievement system and gamification
  • Class reviews and ratings

For Instructors

  • Class management dashboard
  • Student analytics and engagement tracking
  • Revenue tracking with 15% commission model
  • Automated payout system
  • Rating and review management

Getting Started

Prerequisites

  • Xcode 15.0+
  • iOS 16.0+ deployment target
  • Supabase account
  • Stripe account (for payments)

Installation

# Clone the repository
git clone https://github.com/Chromeox/HobbyistSwiftUI.git
cd HobbyistSwiftUI

# Open the Xcode project
open iOS/HobbyistSwiftUI.xcodeproj

# Install dependencies
cd iOS
swift package resolve

Architecture

HobbyistSwiftUI follows the MVVM (Model-View-ViewModel) architectural pattern with dependency injection for better testability and maintainability.

Key Components

  • ServiceContainer for dependency injection
  • Protocol-oriented service design
  • Async/await for network operations
  • Combine for reactive programming

Design Patterns

  • Repository pattern for data access
  • Coordinator pattern for navigation
  • Observer pattern for state management
  • Factory pattern for object creation
