# Scripts Directory

Organized automation and utility scripts for the Hobbyist platform.

## Directory Structure

### `/automation/`
- Web scraping and data automation scripts
- Google Sheets integration
- Airtable synchronization
- Email notifications
- Gemini AI integration

### `/instagram/`
- Instagram data scraping scripts
- Account management utilities
- Content extraction tools

### `/database/`
- Database migration and fix scripts
- UUID correction utilities
- Data standardization tools

### `/utilities/`
- Testing and validation scripts
- App integrity checks
- Monitoring tools
- Setup automation

## Quick Start

Most automation scripts require environment variables for API keys and credentials. Check individual script files for required configuration.

### Common Commands
```bash
# Run automation pipeline
./automation/1-run-now.sh

# Monitor automation
./automation/2-monitor-live.sh

# Test Instagram access
./utilities/test-instagram-access.js
```
