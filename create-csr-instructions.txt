=================================================

STEP 1: Open Keychain Access
----------------------------
1. Press Cmd+Space and search "Keychain Access"
2. Open the application

STEP 2: Request Certificate
---------------------------
1. In Keychain Access menu â†’ Certificate Assistant â†’ Request a Certificate From a Certificate Authority
2. Fill in the form:
   - User Email Address: kurtcuff@gmail.com (your Apple ID email)
   - Common Name: Kurt Cuffy (your name)
   - CA Email Address: LEAVE BLANK
   - Request is: "Saved to disk" (select this option)
   - Let me specify key pair information: CHECK THIS

STEP 3: Key Pair Information
----------------------------
1. Key Size: 2048 bits
2. Algorithm: RSA
3. Click Continue

STEP 4: Save CSR File
---------------------
1. Choose location to save (like Desktop)
2. Name it something like "HobbyistApp_CSR.certSigningRequest"
3. Click Save

STEP 5: Upload to Apple
-----------------------
1. Go back to Apple Developer Console
2. Click "Choose File" and select your CSR file
3. Click Continue

RESULT:
- You'll get a certificate (.cer file) to download
- Double-click to install it in Keychain
- Then you can create your provisioning profile

WHY THIS WORKS:
- Creates a public/private key pair on your Mac
- Sends public key to Apple for certificate creation
- Keeps private key secure on your Mac
- Allows code signing of your apps
